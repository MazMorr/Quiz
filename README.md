# Quiz # JavaFX # SceneBuilder 

This project is for an event when the client needs 30 questions to deploy included images and videos, the program is make for two monitor(one for the public and the other for the guy who is deploying the questions), so they can prepare the program by himselfs to show a wallpaper, media and text for the questions. I except than this hard work can help to somebody

Resources: ApacheNetBeans or Intellij for the Packaging
(go to code and reprogram the path to the archives with the route than you prefer the most! you will see this in the primaryController archive, handleQuestion method for the txt and media. for the wallpaper go to SecondaryController archive, initialize method and there you can change the path for the Wallpaper)
